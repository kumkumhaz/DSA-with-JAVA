CLASS:-
      - User defined blueprint
      - Blueprint/templates of real life/world entity
      - A class defines the properties and behaviors that an object class can have.
OBJECT:- 
      - Instance of a class
      - real world entity

- access modifiers
- Class keyword
- Class name
- Body name
